# 2026-05-02 日志

## 财务管家 v2.3 优化 - 收入记录功能
- **需求来源**：轩哥反馈原来的余额计算太死板（固定预算），需要自己记录每月生活费到账情况
- **新增收入记录弹窗**：`finOpenIncomeSheet()`，支持选来源标签（生活费/奖学金/兼职/红包/其他）、自定义备注、金额快捷按钮（2500/2000/1000/500）、**自选日期**
- **record数据结构新增type字段**：`income`（收入）/ `expense`（支出，默认兼容旧数据）
- **余额计算逻辑改造**：有收入记录→余额=本月收入-本月支出；无收入记录→fallback到预算（向下兼容）
- **总览页更新**：有收入时显示"本月收入"格（绿色），无收入时仍显示"月预算"
- **记录显示区分**：收入显示绿色`+¥`，支出显示红色`-¥`，收入图标💰绿色背景
- **记账页新增收入入口**：绿色渐变按钮"＋ 记一笔收入"，与支出区分
- **连续记账/统计只计支出天数**：排除纯收入日
- **旅游基金**：有收入记录时基于实际收入计算可攒金额
- **新增水果🍎快捷标签**

## 财务管家 v2.3 - 密码锁 + 数据加密 + 分账户
- **密码锁（方案A+B）**：首次打开财务模块设置密码，本设备自动记住（设备指纹），换设备需重新验证
- **数据加密（XOR cipher）**：localStorage中的收支数据和设置用密码加密存储（前缀`enc:`），明文数据自动迁移加密
- **设备绑定**：`_finDeviceId()`基于UA+屏幕+时区生成指纹，首次验证后写入已信任设备列表
- **修改密码**：设置页新增"修改密码"功能，修改后自动用新密码重新加密所有数据
- **分账户**：新增支付宝💙/微信💚/现金💵三个账户，每笔记账可选账户
- **支出弹窗**：底部新增账户选择标签行，记忆上次选择的账户
- **收入弹窗**：新增到账账户选择（默认支付宝）
- **总览分账户余额**：新增🏦卡片，分别显示支付宝/微信/现金的月内余额
- **记录列表**：每条记录名称后显示账户图标badge
- **record新增account字段**：`alipay`/`wechat`/`cash`
